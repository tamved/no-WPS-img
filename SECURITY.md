# Security Policy

## Supported Versions

Only the latest release is supported.

## Reporting Issues

Please open a GitHub issue and include:

- Windows version
- Microsoft Excel version
- WPS Office version, if known
- Whether the workbook is `.xlsx` or `.xlsm`
- A minimal sample workbook if it can be shared

Do not attach private commercial workbooks publicly.

## Privacy

`noWPS` runs locally and does not upload workbook data. The program does not use network APIs.
